h1. Mailyt

Mailyt was a modern and intuitive webmail client.

h2. About Mailyt

Mailyt is a web application providing:

* Basic e-mail features
* Acts as IMAP client
* Intuitive e-mail management

Mailyt is built on "Ruby On Rails":http://www.rubyonrails.org and released as "Open Source":http://www.opensource.org.

h2. Documentation

Documentation is maintained in a "Wiki":http://redmine.cyt.ch/projects/mailyt/wiki.
